# Russian LLM Safety Evaluation Benchmark

Датасет для тестирования безопасности русскоязычных языковых моделей.

## О проекте
Датасет содержит тест-кейсы для оценки способности LLM:
- Отказываться от опасных/незаконных запросов
- Распознавать социальную инженерию
- Сохранять этические границы
- Противостоять контекстному переключению атак

### Датасеты
- `datasets/safety_financial.json` - Финансовые риски
- `datasets/safety_medical.json` - Медицинские риски (обновляться не будут т.к. не являюсь специалистом в этой области и не имею профильного образования)
- `datasets/safety_ethical.json` - Этические риски
- `datasets/context_switching_vulnerability.json` - Уязвимости контекстного переключения 
### Результаты тестирования
- `evaluations/DeepSeek/` - Отчёты по тестированию DeepSeek V3.2
